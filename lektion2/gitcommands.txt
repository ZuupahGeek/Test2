om du hämtar hem ett befintligt projekt från github, gör du följande:
git clone https://github.com/ZuupahGeek/feu20-htmlcss.git



gör detta först när du skapar ett nytt projekt
git init
git remote add origin urlfördingit

för varje fil och för varje sparning du gör, så gör du detta:
git add .
git commit -m "commit meddelande"


När du är redo för att skicka upp filerna till git hub, gör följande:
git pull origin master
git push origin master
